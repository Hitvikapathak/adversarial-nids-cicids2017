## Option A — Fastest (GitHub website, you are already logged in)

### Step 1: Create the repository

1. Open https://github.com/new in Chrome.
2. **Repository name:** `adversarial-nids-cicids2017`
3. **Description:** `Adversarial robustness evaluation for ML-based NIDS on CIC-IDS2017 (IITK B.Cyber project)`
4. Choose **Public**.
5. Do **NOT** add README, .gitignore, or license (they already exist locally).
6. Click **Create repository**.

### Step 2: Push from your computer (PowerShell)

Open PowerShell and run these commands one by one:

```powershell
$env:Path = "C:\Program Files\Git\cmd;" + $env:Path
cd C:\Users\HP\adversarial-nids-cicids2017
git remote add origin https://github.com/hitvika/adversarial-nids-cicids2017.git
git branch -M main
git push -u origin main
```

When prompted, sign in with your GitHub account (browser popup or credential manager).

### Step 3: Verify

Open https://github.com/hitvika/adversarial-nids-cicids2017 — you should see README, `src/`, `results/`, `docs/`, etc.

## What gets uploaded (and what does NOT)

### Uploaded to GitHub

| Folder/File | Purpose |
|-------------|---------|
| `README.md` | Project overview + reproduce steps |
| `LICENSE` | MIT license |
| `requirements.txt` | Python dependencies |
| `run.py` | One-command pipeline |
| `src/` | All source code |
| `scripts/` | Report generation + publish script |
| `notebooks/` | Jupyter notebook guides |
| `docs/` | Reflection + demo script |
| `results/metrics.json` | Exact experiment numbers |
| `results/*.png` | Confusion matrices + graphs |
| `results/*.csv` | Summary tables |
| `data/processed/dataset_profile.json` | Dataset statistics |
| `deliverables/` | Report PDF + one-pager copies |

### NOT uploaded (too large / auto-downloaded)

- `data/raw/*.csv` — 52 MB CIC-IDS2017 file (downloads automatically on first `python run.py`)
- `models/*.joblib` — trained model binaries (regenerated by pipeline)
